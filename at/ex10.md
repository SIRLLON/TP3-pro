# Exercicio 10 - Varredura TCP Controlada em Python

Aqui a gente fez um script bem simples para testar varias portas do proprio computador (localhost) e ver quais delas estao abertas e prontas para receber conexoes.

## Saida no VirtualBox

```
escanendo target: 127.0.0.1
aberta: 22
tempo: 1.98 ms
lista: [22]
```

## Justificativa Simples

A gente detectou a porta `22` aberta, que eh a porta padrao do serviço de SSH (acesso remoto) do Linux na VM. Limitar o intervalo de portas e o tempo maximo de espera (timeout) eh essencial para o scanner terminar rapido. Fazer varreduras muito grandes ou sem controle na internet pode parecer um ataque hacker, fazendo com que sistemas de segurança (firewalls) bloqueiem o nosso computador para sempre.
