# Exercicio 11 - Transferência de Arquivo via Socket TCP

Aqui a gente criou um script para enviar um arquivo do cliente para o servidor. O cliente avisa qual o tamanho do arquivo antes de começar, e depois envia o conteudo dividido em pedacinhos.

## Saida do Servidor no VirtualBox

```
servidor arquivo rodando
tamanho esperado: 5000
recebido: 5000
```

## Saida do Cliente no VirtualBox

```
enviado: 5000
```

## Justificativa Simples

A gente precisa enviar o arquivo em blocos (pedacinhos) para nao sobrecarregar a memoria do computador. Se a gente tentasse ler um arquivo gigante (tipo um filme de 4GB) de uma vez so para mandar pela rede, a memoria RAM iria acabar e o programa iria travar. Dividindo em pedaços pequenos (tipo 4096 bytes), conseguimos enviar arquivos de qualquer tamanho de forma segura e leve.
